Leyes de Exponentes

Descubre el poder de los exponentes con ejemplos claros y explicaciones detalladas

1. Producto de potencias con la misma base

Fórmula: \(a^m \times a^n = a^{m+n}\)

Ejemplo: \(2^3 \times 2^4 = 2^{3+4} = 2^7 = 128\)

2. Cociente de potencias con la misma base

Fórmula: \(\frac{a^m}{a^n} = a^{m-n}\)

Ejemplo: \(\frac{5^6}{5^2} = 5^{6-2} = 5^4 = 625\)

3. Potencia de una potencia

Fórmula: \((a^m)^n = a^{m \cdot n}\)

Ejemplo: \((3^2)^3 = 3^{2 \cdot 3} = 3^6 = 729\)

4. Producto de potencias con el mismo exponente

Fórmula: \((a \times b)^m = a^m \times b^m\)

Ejemplo: \((2 \times 3)^4 = 2^4 \times 3^4 = 16 \times 81 = 1296\)

5. Cociente de potencias con el mismo exponente

Fórmula: \(\left(\frac{a}{b}\right)^m = \frac{a^m}{b^m}\)

Ejemplo: \(\left(\frac{4}{2}\right)^3 = \frac{4^3}{2^3} = \frac{64}{8} = 8\)

6. Exponente cero

Fórmula: \(a^0 = 1\) (para \(a \neq 0\))

Ejemplo: \(7^0 = 1\)

7. Exponente negativo

Fórmula: \(a^{-n} = \frac{1}{a^n}\)

Ejemplo: \(2^{-3} = \frac{1}{2^3} = \frac{1}{8}\)
